OAuth 준비

OAuth provider 설정이 들어오면 아래 버튼으로 외부 로그인 흐름을 바로 확인할 수 있습니다.

실습 준비 전에는 `GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET` 값을 채워야 합니다.

흐름: Google 로그인 -> Google이 사용자 정보 반환 -> 우리 서버가 User 연결 -> JWT 발급 -> 이 페이지로 redirect

비밀번호 재설정 메일 요청

실습 준비 전에는 `MAIL_HOST`, `MAIL_PORT`, `MAIL_USERNAME`, `MAIL_PASSWORD` 값을 채워야 합니다.

흐름: email 입력 -> reset 링크 생성 -> SMTP 메일 발송
